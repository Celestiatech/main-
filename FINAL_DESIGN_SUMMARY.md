## 🎨 Design System Details

### Color Palette
```
Primary:    #3B82F6 (Blue)
Secondary:  #8B5CF6 (Purple)
Success:    #10B981 (Green)
Text Dark:  #1F2937
Text Light: #64748B
```

### Shadow System (3-Layer Approach)
```
Layer 1 (Base):    0 10px 30px rgba(0,0,0,0.08)
Layer 2 (Glow):    0 0 40px rgba(139,92,246,0.1)
Layer 3 (Premium): 0 12px 35px rgba(59,130,246,0.4)
```

### Animation Timing
```
Standard:  0.3s ease
Premium:   0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94)
Smooth:    0.5s ease-in-out
```

### Border Radius Scale
```
Cards:   24px (modern look)
Buttons: 12px
Inputs:  12px
Badges:  20px
Icons:   50% (circular)
```
