# NexaVibe Brand Assets

This directory contains all brand assets for NexaVibe Solutions.

## Required Assets

### Logo Files
- `logo.svg` - Primary logo (SVG format, optimized)
- `logo-dark.svg` - Dark mode variant
- `logo-light.svg` - Light mode variant
- `favicon.ico` - Favicon (16x16, 32x32, 48x48 sizes)
- `social-logo.png` - Social media logo (1200x630px for Open Graph)

### Open Graph Image
- `og-image.png` - Default Open Graph image (1200x630px)

## Logo Specifications

### Primary Logo
- Format: SVG
- Color: Primary brand color (#3b82f6)
- Usage: Header, footer, main branding

### Dark Mode Logo
- Format: SVG
- Color: Light/white variant
- Usage: Dark backgrounds

### Light Mode Logo
- Format: SVG
- Color: Dark variant
- Usage: Light backgrounds

### Favicon
- Format: ICO (multi-size)
- Sizes: 16x16, 32x32, 48x48
- Usage: Browser tab icon

### Social Logo
- Format: PNG
- Size: 1200x630px
- Usage: Social media sharing, Open Graph

## Notes
- All logos should maintain aspect ratio
- SVG logos should be optimized for web
- Ensure logos are readable at small sizes
- Maintain brand consistency across all variants
